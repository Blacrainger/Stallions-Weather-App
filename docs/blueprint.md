# **App Name**: WeatherWise

## Core Features:

- Location Detection: Automatically detect the user's location using their IP address or device location services.
- Weather Data Fetching: Fetch current weather conditions and forecast data from a weather API based on the detected location.
- Data Storage: Store fetched weather data in a MongoDB database using the provided connection string: mongodb+srv://AUN-Web-APP:AUNapp@cluster0.ci5xuq8.mongodb.net/?appName=Cluster0.
- AI-Powered Outfit Recommendation: Based on the current weather conditions (temperature, rain, snow, etc.) and using an AI tool, the application will suggest appropriate clothing options.
- Weather Display: Display the current weather conditions (temperature, humidity, wind speed, etc.) in a clear and user-friendly format on the user interface.
- Forecast Display: Display a short-term weather forecast for the next few hours or days, including temperature, conditions, and precipitation probability.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to reflect clear skies and a fresh feeling.
- Background color: Light gray (#F0F0F0) to provide a clean and neutral backdrop.
- Accent color: Sunset orange (#FFB347) for highlighting important information or interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif, combines a modern look with a touch of warmth for readability on mobile devices.
- Use clear, intuitive icons to represent weather conditions (sun, cloud, rain, snow, etc.).
- Use a responsive layout that adapts to different screen sizes for optimal viewing on mobile devices.
- Subtle animations or transitions to indicate loading states or changes in weather conditions.
